# Description of the project 
Fully offgrid repeater/router working in Meshtastic network. Designed for deployment on the roof of WFAIS UJ or any other remote location. Powered by a solar panel in sunny times and by a battery bank in not so sunny moments. Project includes used 3D models, guidelines for dercreasing power consuption of the setup and mathematica notebook for estimating number of batteries you should use. There are also some photos for inspiration purposes.
